# DBC Data Studio

A Venture Lab for creating multiple streams of passive income with minimal customer interaction.

## Projects

### CatalogHero Smoke Test

**Status:** Active Development

A smoke test for a Shopify Catalog Cleaner focused on transforming commercetools JSON into clean Shopify CSVs.

**Features:**
- JSON-to-CSV Flattener (in development)
- Waitlist collection
- Analytics tracking
- Admin dashboard

**Tech Stack:**
- FastAPI
- Python 3.9+
- Tailwind CSS

**Quick Start:**

```bash
cd catalog-hero-smoke-test
./setup.sh
python main.py
```

Visit `http://localhost:8000` for the landing page.
Visit `http://localhost:8000/admin` for the admin dashboard.

**Deployment:**
Ready for deployment via Docker to Railway, Render, or Google Cloud Run.
