# Paid Traffic Blueprint - Google/Bing Ads Strategy

**Budget:** $10-20/day  
**Strategy:** High-Intent Keywords → Direct to Affiliate Links  
**Goal:** Zero-Touch Conversion Funnel

---

## Campaign Structure

### Campaign 1: High-Intent Search (Primary)
**Daily Budget:** $12-15/day  
**Match Types:** Exact + Phrase Match  
**Focus:** Users actively searching for solutions

### Campaign 2: Brand Awareness (Secondary)
**Daily Budget:** $3-5/day  
**Match Types:** Broad Match (carefully managed)  
**Focus:** Build awareness, retarget later

- `how to build`

## Landing Page Strategy

### Single Landing Page per Niche
**Format:** "Best [Tool Type] 2025 - Top 5 Compared"

**Structure:**
1. Hero: "We tested 20+ tools. Here are the winners."
2. Comparison Table: Top 5 tools side-by-side
3. Detailed Reviews: Each tool with pros/cons
4. Affiliate Links: Clear CTAs to each tool
5. FAQ Section: Common questions

**Key Features:**
- Multiple affiliate links (top 3-5 tools)
- Comparison table for easy scanning
- Mobile-optimized
- Fast loading (< 2 seconds)
- Clear value proposition

## Budget Allocation

### Daily Budget: $15/day (Recommended)

**Breakdown:**
- High-Intent Search: $12/day (80%)
- Brand Awareness: $3/day (20%)

**Monthly Budget:** ~$450
- Allows meaningful data collection
- "Fail Fast" approach
- Can scale or kill quickly

### Scaling Strategy

**Week 1-2:** Test at $10/day
- Gather data
- Identify winning keywords
- Optimize ad copy

**Week 3-4:** Scale winners to $20/day
- Double down on high performers
- Kill underperformers
- Refine targeting

**Month 2+:** Scale based on ROI
- If positive ROI: Scale to $30-50/day
- If negative ROI: Pivot or kill

## Tracking & Metrics

### Key Metrics to Track
- **Click-Through Rate (CTR):** Target > 2%
- **Cost Per Click (CPC):** Monitor by keyword
- **Conversion Rate:** Affiliate link clicks
- **Cost Per Acquisition (CPA):** Affiliate commission vs ad spend
- **Return on Ad Spend (ROAS):** Target > 300%

### UTM Parameters
```
utm_source=google_ads
utm_medium=cpc
utm_campaign=[niche]_[campaign_type]
utm_term=[keyword]
utm_content=[ad_variant]
```

## Risk Mitigation

1. **Budget Control:** Daily budget caps prevent overspending
2. **Negative Keywords:** Prevent irrelevant clicks
3. **Fail Fast:** 2-week test cycles
4. **Multiple Affiliate Links:** Diversify revenue sources
5. **Professional Conflict:** All niches avoid commercetools overlap
